### 日志

使用 seelog 。 dev.xml.tmpl 和 prod.xml.tmpl 是我定制的两个日志文件的配置，一个用于开发环境，一个用于生产环境。

特性：

|     | dev | prod |
| --- | :---: | :---: |
| 输出至控制台                                              | ✔ | ✗ |
| 输出至文件                                                | ✔ | ✔ |
| 日志目录不存在时自动创建                                  | ✔ | ✔ |
| warn级别以上日志除输出至主日志文件外，另外输出至一个文件  | ✔ | ✔ |
| 打印trace、debug级别日志                                  | ✔ | ✗ |
| 控制台日志级别字段多颜色                                  | ✔ | ✗ |
| 时间字段包含毫秒                                          | ✔ | ✗ |
| 打印源码文件及源码行号字段                                | ✔ | ✗ |
| 日志文件按天翻滚                                          | ✗ | ✔ |
